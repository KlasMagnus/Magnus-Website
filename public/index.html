Ongoing projects

Magnus's Handy ToolBox

The Toolbox contains a bunch of Windows applications I wrote to make my workdays easier.
It takes the form of a dashboard and contains time reporting tools connected to Outlook for my consulting work as well as cryptography tools.

It's written in C# as a WPF Application using the MVVM Design Pattern.
The idea to use this pattern was born out of my frustration with WinForms applications and how the GUI code runs a high risk to be very tightly coupled to your business logic. I had a lot of fun learning this and I think it's a shame the WPF style with XAML wasn't more widely adopted. The ToolBox is Open Source and available under the MIT License on my GitHub Page
